## Featured Projects

### 🏠 Automated Residential Inquiry System — RAG Pipeline
> **~40% reduction in irrelevant responses · Sub-2s latency · Serving 1,000+ residents**

A production-ready RAG pipeline built for a real residential community. The system handles document ingestion, semantic chunking (`RecursiveCharacterTextSplitter`), HuggingFace embeddings (`all-mpnet-base-v2`), ChromaDB vector store, and Gemini API generation — implemented with both LangChain toolkits and custom retrieval logic written from scratch.

The engineering challenge wasn't just building it — it was evaluating it. I designed a full AI evaluation framework: 70+ real-user query test cases, quality rubrics, and root cause analysis on retrieval failure modes. Iterating on chunking strategy and prompt design drove a measurable ~40% drop in irrelevant responses.

**Stack:** `Python` `LangChain` `ChromaDB` `Gemini API` `HuggingFace Embeddings` `Streamlit`  
**Deployed:** Hugging Face Spaces + Streamlit Cloud Community

---

### 📄 Unsupervised Text Summarization — Published Research
> **10,000+ documents · ROUGE-evaluated · Published in JETIR Vol. 9 (2022)**

Extractive NLP pipeline built from the ground up: K-Means clustering, TF-IDF feature extraction, and Euclidean distance sentence ranking — implemented in self-developed code alongside scikit-learn. Evaluated against supervised baselines using ROUGE metrics. Findings published in the Journal of Emerging Technologies and Innovative Research.

**Stack:** `Python` `scikit-learn` `NLP` `TF-IDF` `K-Means`  
**Publication:** [JETIR, Vol. 9, 2022](https://www.jetir.org)
