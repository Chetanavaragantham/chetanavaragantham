## 🚀 Featured projects

### 🏠 Automated Residential Inquiry System — RAG Pipeline
> `~40% fewer irrelevant responses` · `sub-2s latency` · `1,000+ residents served`

Production RAG pipeline for a real residential community — document ingestion, semantic chunking (`RecursiveCharacterTextSplitter`), HuggingFace embeddings (`all-mpnet-base-v2`), ChromaDB vector store, Gemini API generation. Built with both LangChain toolkits and custom retrieval logic from scratch.

The hard part was evaluation. I designed a full AI eval framework: 70+ real-user test cases, quality rubrics, and root cause analysis on retrieval failures. Iterating on chunking strategy + prompt design drove the measurable 40% drop.

`Python` `LangChain` `ChromaDB` `Gemini API` `HuggingFace Embeddings` `Streamlit`  
🌐 Deployed on Hugging Face Spaces + Streamlit Cloud

---

### 🤖 AI Code Review Agent — Agentic System
> `ReAct agent` · `GPT-4o-mini` · `27 pytest tests` · `CI/CD with GitHub Actions`

Agentic system that autonomously reviews Python code — the agent decides which static analysis tools to run (AST parsing, cyclomatic complexity, security scanning, style checking), synthesizes the results, and returns a structured review with issues, severity levels, and actionable suggestions.

Engineering highlights: two-pass LLM design for guaranteed structured output, tool call tracking, graceful degradation on tool failure, 25s timeout with valid fallback, FastAPI backend with JSON Lines logging + cost estimation, all LLM calls mocked in tests, and a Gradio UI.

`Python` `LlamaIndex` `OpenAI API` `FastAPI` `Pydantic` `pytest` `Docker` `GitHub Actions` `Gradio`
